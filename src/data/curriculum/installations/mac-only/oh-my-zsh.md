Oh My Zsh is a framework for Z-shell which makes the terminal nicer to work with.

Run the following command in the terminal to install oh-my-zsh:

```shell
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
